Search Valid queries are space-delimited, and include :
  • Gene names (e.g. gcn4, rad52, gal4, yap6)
  • Regular expressions of gene names (e.g. gcn*, rad*, gal*, yap*)
  • GO term accessions (e.g. GO:0006366, GO:0004402)
  • Double quote-enclosed full or partial GO term names (e.g. "DNA binding", "cell cycle", "acetylation", "cytoplasm")
Any combination of the above is valid, however longer and more complex queries will take significantly longer than simple ones.

Regular expressions may match many different gene names across multiple organisms, which is not always desirable. Moreover, regular expressions will significantly slow the search time, so use them wisely (as in the cases given above like yap* and gcn*, where families of genes have been grouped by name).

To find GO term accessions, term names, or simply to learn more about the GO project, please visit the GO homepage.

GO enrichment results The GO enrichment annotations for each model are organized into tables which can be expanded or collapsed by clicking on the ontology header (e.g. the 'Biological Process', 'Molecular function, or 'Cellular Component'). These GO enrichment annotation tables are grouped first by organism, then by ontology. Within each ontology, the annotations are first sorted by the number of matching query terms, then by enrichment P-value.

Models that contain many genes with the same GO annotation were scored for statistical significance using the hypergeometric test. The P-value of an enrichment indicates the significance of the overlap, K, between an annotation with A genes and a model with M genes.

Errors/Warnings Errors and warnings concerning the query are displayed in red here. In particular, when the query matches a gene synonym, the message will have a message in the form:

[query] matches:
    [S], a synonym of G in O. G is shown in results.


S is the gene (S)ynonym matched by query.
G is the standard (G)ene symbol of S.
G is a gene in (O)rganism.
G will be shown and highlighted in the results, instead of S.

In this example, the warning informs us that the query rad* matched the synonyms of several genes. This kind of message gives us a heads up on what names in the results correspond to which gene names that matched the query.
